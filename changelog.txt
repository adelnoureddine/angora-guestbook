Angora Guestbook
Version 1.6.1 -> 2.0


CHANGES
--------

- Big modernization the codebase to work with PHP 8 and MySQL 8 / MariaDB 10
- Add 2 new modern themes: light and dark
- Add feature to display posts by their operating systems or browser
- Add option to sort posts by date, ascending or descending
- Add statistics for ratings
- Add link to openstreetmap for posts' location
- Fix many small papercuts (timezone, phpinfo, style, etc.)
- Update statistics with modern browsers and operating systems
- Update ReCaptcha to support version 2
- Update TinyMCE to version 3.5.12
- Replace dUnzip2 with ZipArchive
